> 所有页面共享 `PixelBackground` 像素粒子背景 + CRT 扫描线覆盖层。
> 样式使用 styled-components，设计令牌引用 `@theme` CSS 变量，详见 `风格指南.md`。

## 2. GameScreen（游戏主界面）

**路由**：`/game`
**组件**：`src/views/GameScreen.tsx`

```
┌──────────────────────────────────────────────────┐
│ [PixelBackground]                                │
│                                                  │
│ ┌──────────────── GameHeader ──────────────────┐ │
│ │ < 密室1/7   ●●○○○○○   ⭐35   ⏱05:32       │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
│ ┌──────────────── 房间场景 (RoomScene) ─────────┐ │
│ │                                              │ │
│ │   ◈1-1        🖥️           ◈1-3🔒           │ │
│ │  (发光脉冲)   (装饰)      (锁定灰色)         │ │
│ │                                              │ │
│ │        📦          🚪(出口)                   │ │
│ │      (装饰)     (锁定/解锁)                  │ │
│ │                                              │ │
│ │           🧑                                  │ │
│ │         (角色)                                │ │
│ │                                              │ │
│ │   ✓1-2                    ◈1-4🔒             │ │
│ │  (绿色已完成)            (锁定灰色)          │ │
│ │                                              │ │
│ └──────────────────────────────────────────────┘ │
│                                                  │
│ ┌─ 底部状态栏 ─────────────────────────────────┐ │
│ │  密室1: 时间档案馆    谜题: 1/4已完成         │ │
│ └──────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────┘
```

**GameHeader 组件**：
- 高度固定 48px，背景 `bg-bg-secondary`，底部 4px `pixel-green` 实线
- 左：返回箭头 + 密室编号
- 中：7 个进度点（完成=绿●, 当前=脉冲●, 未到=灰○）
- 右：分数 `pixel-gold` + 计时器 `text-secondary`

**RoomScene 组件**：
- 占满剩余空间，比例保持 3:2（最大 960×640）
- 使用 react-konva (Canvas) 渲染所有场景元素：装饰物、答题点、出口门、角色
- 角色不再使用 DOM 绝对定位覆盖层，而是直接渲染在 Konva Layer 中
- **坐标系统统一**：所有元素的 (x, y) 坐标代表元素**中心点**（百分比 0-100），通过 `offsetX/offsetY` 实现中心锚点
- 装饰物 (decors) 使用**左上角**锚点（因为有 width/height 区域概念）
- 答题点、出口门、角色使用**中心锚点**（通过 `offsetX/offsetY` 偏移自身尺寸的一半）
- 点击空白区域时，Konva Stage 的 pointerPosition 转换为百分比坐标传给角色移动
- 点击答题点/出口门时，通过 `name` 属性过滤，不触发空白区域移动

## 7. 页面流转关系

```
StartScreen (/)
    │ 点击"开始逃脱"
    ▼
GameScreen (/game)
    │
    ├── RoomEntrance (覆盖层, 每个密室开始时)
    │       │ 点击"进入密室"
    │       ▼
    ├── RoomScene (房间探索, 角色走动)
    │       │ 点击答题点 → 角色走过去
    │       ▼
    ├── PuzzleModal (答题弹窗)
    │       │ 答完关闭, 回到 RoomScene
    │       ▼
    ├── RoomCleared (覆盖层, 密室通关时)
    │       │ 点击"进入下一密室"
    │       ▼
    │   └── 回到 RoomEntrance (下一密室)
    │
    │ 第7密室通关后
    ▼
VictoryScreen (/victory)
```
